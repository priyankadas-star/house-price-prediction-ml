# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts California house prices using Machine Learning. The model is trained on the California Housing Dataset and deployed as an interactive web application using Streamlit.

The application allows users to enter property details such as location, median income, number of rooms, and population to predict the estimated house price instantly.
